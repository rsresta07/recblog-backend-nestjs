      Generate a user-post interaction matrix.

      Parameters

      • users: User[]

        The list of users to generate the matrix for.

      • posts: Post[]

        The list of posts to include in the matrix.

      • postLikes: PostLike[]

        The list of likes on posts, used to calculate interaction scores.

      • comments: Comment[]

        The list of comments on posts, used to calculate interaction scores.

      Returns { [userId: string]: { [postId: string]: number } }

      A matrix where each key is a user ID and the value is an object mapping post IDs to interaction scores.

      Remarks

      This function constructs a matrix where each user ID is mapped to an object that contains post IDs and their interaction scores. The interaction score is calculated based on the number of likes and comments a user has on a given post. Likes and comments are assigned different weights, with likes being weighted more heavily.
